## SBERT-Driven Semantic Trajectory Analysis for Quantifying Technical Buzzword Dilution
1. It detects how technical buzzwords like AI, GPT, and Blockchain are losing their precise meaning as they spread across platforms like LinkedIn, Reddit, and Twitter.

2. It uses Sentence-BERT to convert text into embeddings and measures how far the usage has drifted from the original technical definition using cosine similarity.

3. It introduces two custom metrics — Semantic Trajectory Divergence and Dilution Score — to quantify how much a term has been diluted over time.

4. It analyzed 500+ text samples from 2020–2025 and found that buzzwords follow a three-phase lifecycle: Emergence, Expansion, and Saturation.

5. The results were statistically validated and the system outperforms traditional frequency-based methods in detecting subtle semantic drift.
